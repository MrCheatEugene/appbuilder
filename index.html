Goodbye, desktop!

AppBuilder is an easy way to develop and export web-based mobile applications, right in your browser. It's based on Apache Cordova, that does all of the hard work
AppBuilder has a powerful Monaco editor, with auto-syntax, highlighting and much more. It was created for App Inventor users, who code using Scratch, and who want to level up easily without Java knowledge.

Brain power!

AppBuilder can compile apps for multiple users even on a weak webserver, even with 1 or 2 GB of RAM.

Possibilities

With HTML5, JavaScript, WebAssembly and CSS, you are barely limited. You can create whatever you can on a website, thanks to Chromium engine that lives inside Apache Cordova.

This cloud looks like my yesterday work.

All of your project data is stored on a webserver, so even if you'll accidentally close chrome tab, you can return to yuur project in 2 clicks!

Protected

I mentioned WebAssembly, or WASM multiple times previously. Because it's the best obfuscation method, You can write program in C++, and compile it. With any compiler option that you want to use.
